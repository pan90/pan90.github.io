Visual Studio Code

Visual Studio Code 是一款非常好用的编辑器, 如果你缺少合适的开发工具, 那么我强烈推荐你使用 Visual Studio Code, 它有如下特点:

  • 开源，免费；
  • 自定义配置
  • 集成git
  • 智能提示强大
  • 支持各种文件格式（html/jade/css/less/sass/xml）
  • 调试功能强大
  • 各种方便的快捷键
  • 强大的插件扩展
